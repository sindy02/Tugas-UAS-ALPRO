# Program Pengolahan Data Mahasiswa (Bahasa C)
Program ini adalah aplikasi berbasis bahasa C untuk mengelola dan mengolah data nilai mahasiswa  menggunakan array satu dimensi, array dua dimensi (matriks), serta fungsi rekursif.  
Program berjalan melalui menu interaktif pada terminal/console.
Program ini dibuat untuk memenuhi kebutuhan pembelajaran mengenai:
- Array satu dimensi dan dua dimensi
- Pengolahan data numerik
- Struktur menu dalam bahasa C
- Fungsi rekursif (faktorial dan Fibonacci)
- Pemrograman terstruktur


##  Fitur Program
program menampilkan beberapa menu pilihan sebagi beriku:
  Manajemen Data Mahasiswa
- Menampilkan data mahasiswa
- Menambahkan data mahasiswa
- Mengubah nilai mahasiswa berdasarkan NIM
- Menghapus data mahasiswa berdasarkan NIM
  Pengolahan Nilai (Matriks)
- Menghitung rata-rata nilai tiap mahasiswa
- Menghitung rata-rata nilai Tugas
- Menghitung rata-rata nilai UTS
- Menghitung rata-rata nilai UAS
   Menampilkan mahasiswa dengan:
 - Rata-rata nilai tertinggi
 - Rata-rata nilai terendah
